PLSC 380 Python Course

Welcome to PLSC 380 Python Course. This page contains some of the notebooks that I created for teaching the course at Binghamton University! The course is designed for undergraduates and beginners, and it provides a practical introduction to Python and its key libraries, such as NumPy, Pandas, Matplotlib, Seaborn, Plotly, Geopandas. These codebooks offer a comprehensive and hands-on learning experience.
